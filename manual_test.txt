配置命令:
• /config - 重新配置AI模型
• /status - 显示当前配置状态
• /save - 保存当前配置
• /load - 加载保存的配置

优势:
1. 完全控制: 用户决定何时配置
2. 灵活性: 可以跳过配置，稍后配置
3. 安全性: 不会自动加载可能过期的配置
4. 测试友好: 方便测试不同配置

测试建议:
1. 第一次运行: 选择 'y'，配置DeepSeek API
2. 测试 /chat 功能
3. 退出后再次运行: 选择 'n'，测试未配置状态
4. 使用 /config 命令重新配置
5. 测试 /status 查看配置状态
