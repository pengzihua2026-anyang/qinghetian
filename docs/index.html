关于项目

青禾田 (QingHeTian) 是一个致力于为AI时代打造完全自主、零依赖底层工具链的开源项目。

我们的核心使命是：把AI智能程序开发沉降到二进制，让AI开发回归计算机本质，在二进制层面实现真正的智能控制。

项目采用 开源+商业双许可模式：

  • 🆓 个人使用完全免费 (GPL v3.0许可证)
  • ⏳ 商业授权规划中 (保持灵活性，根据市场需求调整)

技术特性：

  • 🚀 零依赖架构：纯二进制实现，不依赖任何外部库
  • 🤖 AI专用设计：自然语言接口，结构化输出
  • 🔧 完全自主：网络协议、加密算法全部自主实现
  • 📱 高性能优化：针对二进制操作深度优化
