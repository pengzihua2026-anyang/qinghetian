# 二进制编辑器 (Binary Editor)

一个功能完整的命令行二进制文件编辑器，支持十六进制和ASCII模式编辑。

## 功能特性

- ✅ **十六进制显示** - 标准的十六进制转储格式
- ✅ **ASCII显示** - 可读字符显示
- ✅ **双向编辑** - 支持十六进制和ASCII模式编辑
- ✅ **文件操作** - 打开、保存二进制文件
- ✅ **字节操作** - 插入、删除、替换字节
- ✅ **导航功能** - 光标移动、跳转、翻页
- ✅ **搜索功能** - 查找特定字节或字符串
- ✅ **实时预览** - 显示当前字节的十六进制和ASCII值

## 编译安装

### 使用 Makefile (推荐)

```bash
# 编译
make

# 清理
make clean

# 调试版本
make debug

# 安装到系统
make install
```

### 使用 Xcode

```bash
# 打开项目
open binary.xcodeproj

# 或命令行构建
xcodebuild -project binary.xcodeproj
```

## 使用方法

### 基本使用

```bash
# 打开文件
./binary <文件名>

# 示例
./binary test.bin
./binary /path/to/file.exe
```

### 编辑器快捷键

#### 导航
- **↑↓←→** - 移动光标
- **PageUp/PageDown** - 翻页
- **Home** - 跳到文件开头
- **End** - 跳到文件结尾
- **G** - 跳转到指定偏移量

#### 编辑
- **0-9, A-F** - 输入十六进制值
- **Space** - 切换十六进制/ASCII模式
- **I** - 在光标处插入字节 (0x00)
- **D** - 删除光标处字节
- **R** - 替换光标处字节

#### 文件操作
- **S** - 保存文件
- **Q** - 退出编辑器
- **H** - 显示帮助

### 示例操作

1. **查看文件**
   ```
   ./binary example.bin
   ```

2. **编辑字节**
   - 切换到十六进制模式 (Space)
   - 移动到要编辑的位置 (箭头键)
   - 输入两个十六进制数字 (如: 4A)
   - 字节将被修改为 0x4A

3. **插入字节**
   - 移动到插入位置
   - 按 I 插入 0x00 字节
   - 编辑新字节的值

4. **保存文件**
   - 按 S 保存修改
   - 退出时如有未保存修改会提示

## 技术细节

### 文件格式支持
- 任意二进制文件
- 无大小限制 (受内存限制)
- 支持大文件分页显示

### 内存管理
- 动态内存分配
- 文件缓冲区自动扩容
- 修改标记和自动保存提示

### 用户界面
- 终端基于 curses 的界面
- 实时状态显示
- 颜色高亮 (计划中)
