# AI集成二进制编辑器 - 完整指南

## 🎯 项目概述

这是一个**集成AI大模型的二进制编辑器**，支持：
1. **配置AI模型**（地址和Key）
2. **自然语言对话**操作二进制文件
3. **智能解析**和执行二进制操作
4. **学习反馈**机制

## 🚀 快速开始

### 编译和运行

```bash
# 编译AI版本
cd /Users/zihua/Desktop/binary
make ai

# 运行AI版本
./ai_bin_editor
```

### 首次配置

程序启动后会提示配置AI模型：
```
API地址: https://api.openai.com/v1/chat/completions
API密钥: sk-your-api-key-here（你的API密钥）
模型名称: gpt-4（可选）
```

配置会自动保存到 `.ai_binary_config` 文件。

## 📖 完整命令参考

### 系统命令
| 命令 | 说明 | 示例 |
|------|------|------|
| `/config` | 配置AI模型 | `/config <地址> <密钥> [模型]` |
| `/save` | 保存配置 | `/save config.txt` |
| `/load` | 加载配置 | `/load config.txt` |
| `/status` | 显示配置 | `/status` |
| `/chat` | 与AI对话 | `/chat 这个文件是什么？` |
| `/ask` | 询问文件问题 | `/ask 如何修改这个文件？` |
| `/history` | 查看对话历史 | `/history` |
| `/clear` | 清除历史 | `/clear` |
| `/help` | 显示帮助 | `/help` |
| `/exit` | 退出程序 | `/exit` |

### 自然语言命令
| 类型 | 示例 | 解析结果 |
|------|------|----------|
| **查看** | `查看 test.bin 文件` | AI_OP_VIEW |
| **编辑** | `在偏移0x10处修改为0xFF` | AI_OP_EDIT |
| **插入** | `在开头插入0x00字节` | AI_OP_INSERT |
| **删除** | `删除偏移0x20处的字节` | AI_OP_DELETE |
| **搜索** | `搜索48 65 6C 6C 6F模式` | AI_OP_SEARCH |
| **分析** | `分析文件类型` | AI_OP_ANALYZE |
| **创建** | `创建新的二进制文件` | AI_OP_CREATE |
| **导出** | `导出前100字节` | AI_OP_EXPORT |

## 🔧 技术架构

### 系统架构
```
用户输入 → 自然语言解析 → AI模型 → 二进制操作 → 文件输出
      ↑          ↓           ↓           ↓         ↓
   对话界面   关键词匹配   API调用   编辑器核心   文件系统
      ↑          ↓           ↓           ↓
   历史记录   学习反馈   错误处理   结果返回
```

### 文件结构
```
binary/
├── ai_integration.h    # AI集成头文件
├── ai_integration.c    # AI集成实现
├── ai_main.c          # AI版本主程序
├── hex_editor.h       # 二进制编辑器核心
├── hex_editor.c       # 编辑器实现
├── ai_interface.h     # AI命令行接口
└── ai_interface.c     # 接口实现
```

### 数据流
1. **输入解析**：自然语言 → 结构化命令
2. **AI处理**：命令 → AI模型 → 操作建议
3. **执行操作**：操作建议 → 二进制编辑器
4. **结果反馈**：执行结果 → 用户/AI学习
